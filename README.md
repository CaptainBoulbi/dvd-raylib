# DVD-raylib

DVD player animation.
Initiation to [raylib](https://github.com/raysan5/raylib).

# building

You need to install [raylib v5](https://github.com/raysan5/raylib/releases/tag/5.0), but any version should work.

```console
$ make
```

to run

```console
$ ./dvd
```

or

```console
$ make run
```
